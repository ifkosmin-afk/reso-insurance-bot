# 🤖 Telegram-бот РЕСО-Гарантия для страховых агентов

Интеллектуальный ассистент для страховых агентов РЕСО-Гарантия.  
Отвечает на вопросы о продуктах, тарифах и переговорных техниках на основе базы знаний компании.

---

## 📋 Что умеет бот

- Отвечает на вопросы агентов по страховым продуктам РЕСО-Гарантия
- Ищет информацию в базе знаний (BM25-поиск по 5000+ фрагментам)
- Формулирует ответы с конкретными цифрами, условиями и исключениями
- Честно сообщает, если ответа нет в базе знаний
- Поддерживает команды `/start`, `/help`, `/products`

## 📚 База знаний включает

| Продукт | Описание |
|---|---|
| Премиум-защита | НС, до 10 млн руб., 18–75 лет |
| Подорожник | НС, 4 варианта, скидка для клиентов РЕСО |
| Базовая защита | НС, 4 варианта, 500–2000 руб. |
| Активная защита | НС для ЗОЖ и спорта, 1–75 лет |
| НС Пассажир | К ОСАГО/КАСКО |
| Домовой | Страхование квартир (коробочный и классический) |
| Ипотека | Доп. полис к Ипотеке / Заёмщику Сбербанка |
| Переговорные листы | Скрипты продаж НС от ОСАГО и КАСКО |

## 🚀 Установка и запуск

### Шаг 1. Получите токен Telegram-бота

1. Откройте Telegram и найдите **@BotFather**
2. Отправьте команду `/newbot`
3. Введите имя бота (например: `РЕСО Агент`)
4. Введите username (например: `reso_agent_bot`)
5. Скопируйте полученный **токен** (вида `123456789:AABBccdd...`)

### Шаг 2. Установите зависимости

```bash
pip3 install python-telegram-bot openai rank_bm25
```

### Шаг 3. Подготовьте файлы

Убедитесь, что в папке `reso_bot/` находятся:
- `База_1.txt` — `База_5.txt` (база знаний)
- `bot.py` (основной скрипт бота)
- `build_index_bm25.py` (построение индекса)
- `start.sh` (скрипт запуска)

### Шаг 4. Постройте индекс (один раз)

```bash
cd reso_bot
python3 build_index_bm25.py
```

Это создаст файл `reso_bm25.pkl` (~12 МБ) — индекс поиска по базе знаний.

### Шаг 5. Запустите бота

```bash
# Установите токен
export TELEGRAM_TOKEN="ваш_токен_от_BotFather"

# Также нужны переменные OpenAI (уже настроены в среде Manus)
# export OPENAI_API_KEY="..."
# export OPENAI_API_BASE="..."

# Запуск
./start.sh
```

Или напрямую:
```bash
TELEGRAM_TOKEN="ваш_токен" python3 bot.py
```

## ⚙️ Как работает бот (технически)

1. **Агент** задаёт вопрос в Telegram
2. **BM25-поиск** находит 6 наиболее релевантных фрагментов из базы знаний (5188 фрагментов)
3. **LLM (Claude Haiku)** формулирует точный ответ на основе найденных фрагментов
4. Если ответа нет в базе — бот честно сообщает об этом

## 🔄 Обновление базы знаний

Для добавления новых документов:
1. Добавьте текстовые файлы в папку `reso_bot/`
2. Обновите имена файлов в `build_index_bm25.py` (или переименуйте в `База_6.txt` и т.д.)
3. Пересоберите индекс: `python3 build_index_bm25.py`
4. Перезапустите бота
